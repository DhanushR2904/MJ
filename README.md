# Smart Navigation System (Smart_Nav)

An interactive floor-plan navigation system designed for the APJ Block. This application provides a fluid digital experience for navigating building layouts, searching for rooms, and locating faculty members.

## Features

- **Interactive Maps**: Navigate through different floors with a responsive SVG-based floor plan.
- **Fast Search**: Quickly find rooms and faculty members using an optimized search system.
- **Room Details**: View detailed information, including descriptions and directions for each room.
- **Faculty Directory**: Easily locate faculty members and their respective offices.
- **Architect Mode**: Tools for administrators to manage and update floor layouts.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DhanushR2904/MJ.git
   ```

2. Install dependencies:
   ```bash
   cd Major_Project
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components`: React components for the UI, including maps and search.
- `src/data`: Core data for floor layouts and search indexing.
- `api`: Backend API for layout persistence (if applicable).
